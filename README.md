# Summar

## What is Summar?

A mobile application that stores user's bookmarks and uses AI to summarise them
for a quick and easy reading experience.

## Features

- Manage bookmarks
- View bookmarks in-app
- Summarise bookmarks using [Smmry API](https://smmry.com/api)
